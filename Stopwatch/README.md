# Stopwatch App

A simple Stopwatch built with React.

## Features

* Start the stopwatch
* Reset the stopwatch
* Displays time in:

```
HH : MM : SS : MS
```

## Technologies Used

* React
* JavaScript
* CSS

## How to Run

1. Clone the repository.
2. Install dependencies.

```bash
npm install
```

3. Run the project.

```bash
npm run dev
```

## Project Structure

```
src
├── App.jsx
├── App.css
└── main.jsx
```

## Preview

* Click **Start** to begin the timer.
* Click **Reset** to stop the timer and set it back to `00:00:00:00`.
