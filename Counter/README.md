# Counter App

A simple React Counter application.

## Features

* Increase the count
* Decrease the count
* Built using React and useState

## Technologies Used

* React
* JavaScript
* CSS

## How to Run

1. Clone the repository.
2. Install dependencies.

```bash
npm install
```

3. Start the project.

```bash
npm run dev
```

## Project Structure

```
src
├── App.jsx
├── App.css
└── main.jsx
```

## Preview

* Click **Add** to increase the count.
* Click **Remove** to decrease the count.
